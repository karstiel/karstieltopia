Si sabes un poco sobre programaci\u00f3n, quiz\u00e1s sepas que la tradici\u00f3n es que cuando escribes tu primer programa es decir hola mundo as\u00ed que… Di Hola Mundo<\/strong>
\n
\nPersonalmente creo que es esto es as\u00ed porque somos gente muy educada y nos gusta decirle a todo el mundo que pronto habr\u00e1 un nuevo programador aqu\u00ed. De acuerdo, ahora es el momento de escribir nuestro primer programa. \u00bfEst\u00e1s nervioso? No te preocupes, podr\u00e1s hacerlo (Pero recuerda que no tienes que darle de comer despu\u00e9s de medianoche… que luego habr\u00eda un mont\u00f3n de problemas):<\/p>\n

class HolaMundo {\r\n  public static void main (String args[]) {\r\n    System.out.println (\"\u00a1Hola Mundo!\");\r\n  }\r\n}<\/pre>\n
Vale, este es nuestro primer programa y, como puedes ver, hay unas pocas l\u00edneas que ahora mismo quiz\u00e1s sean un poco dif\u00edcil de entender pero, cr\u00e9eme, es muy sencillo.<\/p>\n
En futuras entradas hablaremos en detalle de cada l\u00ednea, pero aqu\u00ed y ahora solamente vamos a decir a grosso modo que has creado una clase que se llama HolaMundo y que la \u00fanica cosa que hace es una acci\u00f3n que se llama main… y \u00bfqu\u00e9 es lo que haces con esta acci\u00f3n llamada main? Pues decir \u00a1Hola Mundo! (ya s\u00e9 que todo esto parece magia negra).<\/p>\n
B\u00e1sicamente est\u00e1s creando una clase con un\u00a0m\u00e9todo\u00a0que se llama directamente porque es el main (cantidad a raudales de magia negra aqu\u00ed).<\/p>\n
Un m\u00e9todo es una colecci\u00f3n de comandos y, en este caso, hay solamente un comando, que es un comando de imprimir (print). Este comando “print” nos mostrar\u00e1 algo en la salida del programa, es decir, en la ventana donde lo has ejecutado, que ser\u00e1 normalmente un terminal.<\/p>\n
Vale, has le\u00eddo estos \u00faltimos p\u00e1rrafos y lo m\u00e1s probable es que si no sab\u00edas nada de programaci\u00f3n sigas sin entender nada. No te preocupes, hablaremos de cada cosa con m\u00e1s detenimiento.<\/p>\n
