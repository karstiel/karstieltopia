Y no precisamente al comecocos, sino al gestor de paquetes de Archlinux.<\/p>\n

Por defecto, el gestor tiene los repositorios base de Archlinux, lo cu\u00e1l puede ser que os vaya ligeramente lentos (todo el mundo usando a los pobres…), as\u00ed que pod\u00e9is buscar una alternativa y para ello, esta distribuci\u00f3n nos proporciona un script en python que lo automatiza.<\/p>\n

    <\/p>\n
  1. Si no ten\u00e9is instalado el python (por ejemplo en la instalaci\u00f3n b\u00e1sica), ten\u00e9is que instalarlo con lo siguiente:\n
    pacman -Sy python<\/div><\/div>\n<\/li>\n

    <\/p>\n

  2. Una vez instalado, lo mejor es crear una copia de seguridad del archivo, por si algo fuera mal, as\u00ed que hac\u00e9is un\n
    <br>cp \/etc\/pacman.d\/mirrorlist \/etc\/pacman.d\/mirrorlist.old<\/div><\/div>\n<\/li>\n

    <\/p>\n

  3. Y ahora a la fiesta de ejecutar el script:\n
    <br>rankmirrors \/etc\/pacman.d\/mirrorlist.old > \/etc\/pacman.d\/mirrorlist<\/div><\/div>\n<\/li>\n

    <\/p>\n

  4. Actualiz\u00e1is con:\n
    <br>pacman -Syu<\/div><\/div>\n<\/li>\n
