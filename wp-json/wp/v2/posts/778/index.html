Es decir, algo normal ser\u00eda:<\/p>\n

<br><br>ArrayList <e> element[] = new ArrayList <e> [size];<br><br><\/pre>\n
Pero ante \u00e9sto Java dice que tururu, as\u00ed que toca buscarse alguna alternativa. Y resulta que aqu\u00ed<\/a> he visto un par de soluciones posibles:<\/p>\n
<br><br>ArrayList <e> [] element = new ArrayList[size];<br><br><\/pre>\n
\u00f3<\/p>\n
<br><br>ArrayList <e> [] element = (ArrayList <e> [])new ArrayList[size];<br><br><\/pre>\n
Estas soluciones dan warnings, pero parece que funcionan correctamente ;)<\/p>\n
