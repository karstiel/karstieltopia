Un apunte r\u00e1pido para montar una carpeta compartida en Windows en el arranque de Linux mediante samba.<\/p>\n

    <\/p>\n
  1. Lo siguiente hay que hacerlo como root.<\/li>\n

    <\/p>\n

  2. A\u00f1adir al archivo \/etc\/fstab<\/strong> la siguiente l\u00ednea (es una sola l\u00ednea):
    cifs credentials=\/root\/.smbcredentials, dmask=0777, fmask=0777 0 0<\/li>\n

    <\/p>\n

  3. C\u00f3mo pod\u00e9is ver en la l\u00ednea anterior, creamos el archivo .smbcredentials<\/strong> en el home de root y ponemos:
    username=
    password=<\/li>\n

    <\/ol>\n
