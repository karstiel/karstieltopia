Un recordatorio r\u00e1pido de como enviar ficheros por ssh.<\/p>\n

La primera forma ser\u00eda la siguiente:<\/p>\n

scp origen destino<\/i><\/b>
Ejemplo: <\/p>\n

scp user@hellohost:\/home\/user\/pillame.tar.gz \/home\/test<\/div><\/div>\n

Que copia del host hellowhost en el directorio \/home\/user con el usuario user el fichero pillame.tar.gz<\/p><\/blockquote>\n

La segunda forma ser\u00eda un poco m\u00e1s rebuscada (supongo que estamos colocados en el origen):<\/p>\n

cat fichero_origen <\/i>| ssh destino<\/i> “cd directorio<\/i>; cat > fichero_destino<\/i>“<\/b>
Ejemplo: siguiendo lo anterior, si queremos volver a dejar el archivo pillame.tar.gz<\/i> en el mismo sitio d\u00f3nde lo pillamos, ser\u00eda algo as\u00ed:<\/p>\n

cat pillame.tar.gz | ssh user@hellohost "cd \/home\/user; cat > pillame.tar.gz"<\/div><\/div>\n
