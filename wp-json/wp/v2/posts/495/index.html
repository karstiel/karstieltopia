Si quieres instalar Skype en Ubuntu 13.10, te encontrar\u00e1s con que, de primeras, no est\u00e1 en los repositorios con los que se instala de cero, pero si a\u00f1ades los repositorios de socios de Canonical, ya aparece.<\/p>\n

Una vez instalado, si intentas ejecutarlo, es muy probable que no funcione. Si pasara esto, vamos a ver el motivo:<\/p>\n

skype<\/pre>\n
Si hay un error de libGL.so.1, podemos ejecutar lo siguiente para ver si puede funcionar:<\/p>\n
LD_PRELOAD=\/usr\/lib\/i386-linux-gnu\/mesa\/libGL.so.1 skype<\/pre>\n
\u00bfHa funcionado? Si es que s\u00ed, vamos a dejarlo para poder lanzarlo de una manera m\u00e1s normal. Editamos el siguiente archivo (o lo creamos si no existe):<\/p>\n
sudo vi\u00a0\/etc\/ld.so.conf.d\/skype.conf<\/pre>\n
Grabamos y ejecutamos<\/p>\n
sudo ldconfig -v<\/pre>\n
Con esto, ya podr\u00edamos arrancar Skype.<\/p>\n
